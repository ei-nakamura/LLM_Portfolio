- インバウンド観光客向けレストラン/文化体験レコメンドサービス(Claude)  
  https://claude.ai/public/artifacts/95af6097-b3ea-4b6c-a225-b8a4a570bad3

  こちらは病院と薬局の待ち時間中に作成しました。
